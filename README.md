# AAR Sa Xarala 🔐

**AAR Sa Xarala** est un gestionnaire de mots de passe sécurisé pour protéger vos secrets numériques.

Le nom est wolof :  
- **AAR** = protéger  
- **Sa Xarala** = ton réseau  

---

## Fonctionnalités

- Stockage sécurisé des mots de passe
- Chiffrement AES
- Génération de mots de passe forts
- Authentification avec mot de passe maître
- Interface moderne (React + Tailwind)

---

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
- Cryptographie AES

---

## Installation

```bash
git clone https://github.com/TON_USERNAME/aar-sa-xarala.git
cd aar-sa-xarala
npm install
npm run dev


#####Build & EXE
